This readme describes the MODIS swath-to-grid tools package developed at
NSIDC. The highest level programs in this package consist of the following
three perl scripts:

mod02.pl - reads MOD02 level1b swath data and MOD03 ancillary swath data
mod10_l2.pl - reads MOD10_L2 snow-cover swath data
mod29.pl - reads MOD29 sea-ice swath data
