MS2GT Tutorials

The best way to learn how to use MS2GT is to work through some examples.

In each of the following tutorials, you will need to order some MODIS data using a tool such as the Earth Observing System (EOS) Data Gateway (EDG). Detailed instructions for using the EDG are not provided in the tutorials found here; however a tutorial for using the EDG is available online.

The MS2GT software uses the mapx library to define grids for various map projection. This library reads two kinds of text files called *.gpd *.mpp files that define a grid and a map projection, respectively. The *.gpd and *.mpp files used in the tutorials can be found in the ms2gt/grids directory. If you followed the procedures under Setting up the Environment, then you should have an environment varialble called $PATHMPP defined that points to this directory. To verify that $PATHMPP has been set correctly, type:

ls $PATHMPP

You should see a directory listing showing several *.gpd and *.mpp files.

Each tutorial has a corresponding directory under ms2gt. These directories contain the text files (other than the *.gpd and *.mpp files) referred to in the particular tutorial to save you the trouble of typing them in.
